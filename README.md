# CodeAlpha_StockPortfolioTracker
#Description
This is a simple console based stock portfoilio tracker which keeps a track of stocks purchased and the total investment of the user in stocks
#Features
The user enters the number of stocks to purchase
User then selects stock name and quantity from the available given stocks
The program keeps track of stocks and its quantity in user's portfolio
The program also tells about the total investment done by user
